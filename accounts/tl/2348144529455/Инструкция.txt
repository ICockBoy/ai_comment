Либо с официального сайта - https://www.softpedia.com/get/Internet/Download-Managers/MEGA-Link-Downloader.shtml

Нажимаем Add links, добавляем ссылки из купленного текстового файла.
Далее в окне где появились файлы правой кнопкой мыши - Select All Single file
Во вкладке "Settings - Download Directory" указываем папку куда будут качаться все файлы.

ИНСТРУКЦИЯ ДЛЯ TDATA
-------------------------------

В каждую папку помещаем 'Telegram.exe' и запускаем аккаунт/ы.
Актуальный 'Telegram.exe' по ссылке с официального сайта - https://desktop.telegram.org/

ИНСТРУКЦИЯ ДЛЯ SESSION
-------------------------------

Нужен специальный софт, в который их можно загрузить.
К примеру TG-giant и другие, которые поддерживают данный тип файлов.
